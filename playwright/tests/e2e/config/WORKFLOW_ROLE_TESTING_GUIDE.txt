IMPACT E2E QUICK GUIDE
======================

Purpose
-------
Use workflow -> role -> testing type -> browser combination config for test planning.
This guide follows existing config style and does NOT change your current env setup.

Config File
-----------
tests/e2e/config/execution-matrix.config.json

Selection Order
---------------
1) workflow
2) role
3) testingType
4) browser

Workflows
---------
- author-editor-collator
- editor-author-collator

Roles
-----
- Author
- Co-Author
- Editor
- Co-Editor
- Collator
- Editor2

Testing Types
-------------
- comment-basic   -> tests/e2e/insert-comment-basic.spec.js
- query-regression -> landing-to-editor-query + query-workflow + query-comment
- track-full      -> tracked_changes + cjk-validator
- all-basic       -> landing-editor-module-basic + insert-comment-basic
- all-regression  -> basic + editor6 + query/comment suites

Browsers (as requested)
-----------------------
- chrome  -> Chromium project (supported)
- firebox -> alias of Firefox (supported)
- edge    -> Chromium with msedge channel (supported)
- opera   -> custom executablePath required (mapped through chromium)
- brave   -> custom executablePath required (mapped through chromium)

How to Start (current scripts)
------------------------------
- Comment basic:
  npm run test:insert-comment:basic

- Matrix-based run (new):
  npm run test:matrix -- --workflow <workflow> --role <role> --testingType <testingType> --browser <browser>

- Matrix dry run (preview command only):
  npm run test:matrix:dry -- --workflow <workflow> --role <role> --testingType <testingType> --browser <browser>

- Basic workflow:
  npm run test:workflow:basic

- Regression workflow:
  npm run test:workflow:regression

- Full workflow:
  npm run test:workflow:full

Reports
-------
Playwright already generates:
- HTML report
- JSON report

Location pattern:
tests/reports/YYYY_MM_DD/local_HHMMSS/

Example profile naming
----------------------
{workflow}.{role}.{testingType}.{browser}

Examples:
- author-editor-collator.Author.comment-basic.chrome
- editor-author-collator.Editor.query-regression.edge

Matrix command examples:
- npm run test:matrix -- --workflow author-editor-collator --role Author --testingType comment-basic --browser chrome
- npm run test:matrix -- --workflow editor-author-collator --role Editor --testingType query-regression --browser edge
- npm run test:matrix -- --workflow author-editor-collator --role Collator --testingType all-regression --browser firebox

Notes
-----
- For Opera/Brave execution, add custom Playwright project with executablePath.
- Keep logoutSafely in afterAll for editor-related suites to avoid stale sessions.
- Optional: add --use-project if your Playwright config includes that browser project.
